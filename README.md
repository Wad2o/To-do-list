# Todo List React
 une app qui vous permet d'organiser et de  planifier votre quotidien en notant vos tâches et en les supprimant une fois effectué !

---

## Fonctionnalités

- Ajouter une tâche avec texte et jour associé  
- Afficher les tâches regroupées par jour de la semaine  
- Supprimer une tâche en un clic  
- Sauvegarde automatique des tâches dans le navigateur  

---

## Basée sur du

- React (Hooks : `useState`, `useEffect`)  
- JavaScript (ES6+)  
- CSS  
- localStorage
